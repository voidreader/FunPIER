Platformer PRO LE
Version: 1.1.3 LE

Welcome to Platformer PRO lite edition use the menu for help -> Assets/Platformer PRO/Show Welcome

The lite edition contains the same base platform and movement code as Platformer PRO but does not  include melee combat, 
advanced enemy options, nor advanced movements such as Ropes, Ledge Climb, Grappling Hook, Stair Walking, 360 Loops, and so on.

Upgrade at: https://www.assetstore.unity3d.com/en/#!/content/43863
